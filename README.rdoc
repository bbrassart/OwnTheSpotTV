Getting Started:

- Git clone
- Bundle install
- Create database and seed it.
- To run all the tests, run "rspec".
- Initialize the rails server.

- Test account
    Username: "dummy"
    Email: "dummy@dummy.com"

- You are good to go!


Note that to upload video, you'll have to create a new account on the production website and authorize OwnTheSpot.TV to access your Instagram media.
